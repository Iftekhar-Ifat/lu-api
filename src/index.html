Leading University API
The Leading University API is a REST API that provides access to data scraped from the Leading University's website. The data is cached, ensuring continuous access even if the Leading University's website is unavailable or takes more than 7 seconds to load.

Get Faculty Information

Get the information of the faculty by providing the department name as a URL parameter.

Endpoint:

GET /api/faculty/:department

Example Request:

GET http://localhost:8080/api/faculty/cse

Get Public Student Result

Get the result of a student by providing the student ID as a URL parameter.

Endpoint:

GET /api/result/:student_id

Example Request:

GET http://localhost:8080/api/result/2012020106

Example Response:

                    {
                        "result": {
                          "public": {
                            "student_id": "2012020106",
                            "semester": "Spring-2020",
                            "cgpa": "3.76",
                            "grade": "A",
                            "credit_completed": "116",
                            "name": "Md. Mahdi Hossain Hira",
                            "program": "B.Sc. (Honours) in Computer Science & Engineering",
                            "department": "Computer Science & Engineering"
                          },
                          "private": {}
                        }
                      }
